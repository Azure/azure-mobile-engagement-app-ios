Make sure the app is opened and then in the Azure Mobile Engagement Portal:

  1. Go to the "Reach" tab, and then select "Data Pushes".
  2. Click on "New data push".
  3. Fill the following values in the form:
    • Name: Provide the name of the campaign to identify it.
    • Category: Leave it as blank with no values.
    • Content data type: Select "Text"
    • Content of your data: { "isDiscountAvailable":true, "discountRateInPercent":10 }
  4. Finally, either send the test notification or create and activate the campaign. You will see the product discount applied when you navigate to the Product discount screen by clicking the "Silent Data Push Notification" button. You can also keep the Product discount screen open and see the discount getting applied as the data push is activated.

Make sure the app is opened and then in the Azure Mobile Engagement Portal:

  1. Go to the "Reach" tab, and then select "Announcements".
  2. Click on "New Announcement".
  3. Fill the following values in the form:
    • Name: Provide the name of the campaign to identify it.
    • Category: Provide the value as "WEB_ANNOUNCEMENT"
    • Delivery time: "In-app only"
    • Title & Message: Provide the title and the message of the initial notification which the user clicks to go to the web announcement.
    • Type of the announcement: Select "Web announcement"
    • Action URL: Optionally specify this as "engagement://demo/videos" that corresponds to the native screen to open by clicking on the "Action button".
    • Title & Announcement: Provide the title and the HTML content of the interstitial notification.
    • Action button text : Provide a value of "OK" which will redirect to a native screen of the app (videos)
    • Exit button text: Provide a value of "Cancel" which will cancel the notification.
  4. Finally, either send the test notification or create and activate the campaign to see the notification.
